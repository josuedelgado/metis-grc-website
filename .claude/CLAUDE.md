## Project: Metis GRC Corporate Website

Corporate/marketing website for Metis GRC — a Cybersecurity GRC (Governance, Risk, Compliance) SaaS platform. The app is live at app.metis-grc.com. This website is the public-facing marketing site.

### Tech Stack
- Astro 5 (static-first, React islands for interactivity)
- React 19 + TypeScript (strict)
- Tailwind CSS v4 (via @tailwindcss/vite)
- framer-motion (scroll animations via AnimatedSection component)
- lucide-react (icons, same set as the app)
- Astro content collections (blog + legal pages in markdown)

### Design Tokens (match the app exactly)
- Primary: #1C4E80 (Deep Blue), Secondary: #4CB2FF (Cyan)
- Success: #10B981, Warning: #F59E0B, Critical: #EF4444
- Background: #F5F7FA, Dark: #111827, Border: #E5E7EB
- Text: primary #111827, muted #6B7280
- Fonts: Inter (sans), Roboto Mono (mono)
- Card radius: 12px, shadows: card / card-hover

### URLs & Config
- Domain: metis-grc.com (corporate) / app.metis-grc.com (GRC app)
- Deploy: GCP Cloud Run, europe-west1
- Contact: mailto to sales@metis-grc.com (no backend)
- Analytics: Google Analytics (GA ID: G-XXXXXXXXXX — replace when ready)
- Cookie consent: EU-compliant banner, defaults to denied, GA only on accept
- i18n: EN (default) + PT, prefix-other-locales strategy
- Sister project: ../metis-grc/metis-grc/ (the app codebase)

### Key Paths
- `src/layouts/BaseLayout.astro` — main layout (GA, OG, JSON-LD, cookie consent)
- `src/lib/constants.ts` — SITE config (name, url, appUrl, contactEmail), NAV_LINKS
- `src/components/layout/Header.astro` — sticky nav, MG logo, Product dropdown, mobile menu
- `src/components/layout/Footer.astro` — 4-col footer, dark bg, language switcher
- `src/components/layout/CookieConsent.astro` — EU cookie banner
- `src/components/shared/PageHero.astro` — reusable hero section (dark/light)
- `src/components/shared/CTASection.astro` — reusable CTA banner
- `src/components/shared/AnimatedSection.tsx` — framer-motion fade-in-up wrapper
- `src/components/home/FrameworkComparison.tsx` — React island comparison table
- `src/content/` — blog posts + legal pages (Astro content collections)
- `src/content.config.ts` — collection schemas (blog: title/desc/date/author/category, legal: title/lastUpdated)
- `src/i18n/en.json` + `src/i18n/pt.json` — translation strings
- `Dockerfile` — multi-stage build (node:20-alpine → serve static)
- `cloudbuild.yaml` — GCP Cloud Build pipeline
- `.github/workflows/ci.yml` — GitHub Actions CI (astro check + build)

### Commands
- `npm run dev` — dev server at localhost:4321
- `npm run build` — production static build to dist/ (15 pages)
- `npm run preview` — preview production build
- `npx astro check` — type checking
